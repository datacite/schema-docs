13. Size
====================

**Obligation:** Optional

**Occurrences:** 0-n

**Definition:** Size (e.g., bytes, pages, inches, etc.) or duration (extent), e.g., hours, minutes, days, etc., of a resource.

**Allowed values, examples, other constraints:**

Free text.

Examples: "15 pages", "6 MB", “45 minutes”

.. rubric:: Example XML

.. code:: xml

  <sizes>
    <size>15 pages</size>
    <size>1 MB</size>
  </sizes>
